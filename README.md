go-jlu-drcom-client
===
A jlu drcom client implemented in Go programming language.

## Quick start
```
cd $GOPATH/src
git clone git@github.com:wucongyou/go-jlu-drcom-client.git
cd go-jlu-drcom-client/cmd
go build
./cmd -conf `pwd`/drcom-config-example.toml
```
